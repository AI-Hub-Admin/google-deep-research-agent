# google-deep-research-agent
Google Search Deep Research Agent to conduct multi turns search calling Google Custome Search API and conduct
deep research to answer your question.

[Live](https://derekzz.aiagenta2z.com/google-deep-research-agent/mcp) on aiagenta2z.com deployed.

### Install

```
pip install requirements.txt
```


#### Running Server

```
uvicorn server:app 
```
